# 전시 게시물 업로드 후 작품 추가 UX 경로 점검

## 1. 현재 진입 경로 (작품 추가 액션으로 가는 경로)

| 출발지 | 경로 | 도착 |
|--------|------|------|
| 전시 **새로 만들기** 완료 | `router.push(\`/my/exhibitions/${id}/add\`)` | **작품 추가** 페이지 |
| 전시 **상세** (`/my/exhibitions/[id]`) | "작품 추가" 버튼 → `/my/exhibitions/[id]/add` | **작품 추가** 페이지 |
| 전시 상세 | "관리자 초대" 버튼 → `/my/exhibitions/[id]/add#invite` | 작품 추가 페이지 (invite 섹션으로 스크롤) |
| **업로드** 탭 "전시 만들기" | `/upload/exhibition` → redirect `/my/exhibitions/new?from=upload` | 새 전시 만들기 → 저장 시 위와 동일하게 `/add`로 이동 |
| 위임 수락 후 (전시 scope) | `/my/exhibitions/[id]/add` 또는 delegation 상세에서 링크 | 작품 추가 페이지 |
| 싱글 업로드 시 전시에 곧바로 넣기 | `/upload?addToExhibition=전시ID&...` → 업로드 완료 시 `router.push(\`/my/exhibitions/${id}\`)` | 전시 **상세** (add 아님) |

정리하면, **“이미 만든 전시에 작품을 더 넣고 싶을 때”** 유저가 갈 수 있는 곳은 **한 군데**다.  
→ **`/my/exhibitions/[id]/add`** (작품 추가 전용 페이지)

---

## 2. 작품 추가 페이지 (`/my/exhibitions/[id]/add`) 내부 로직

### 2.1 단계 구조

- **스텝 1: 참여 작가 (1. exhibition.stepArtists)**  
  - “참여 작가”를 **검색해서 선택** (여러 명 선택 가능).  
  - **외부 작가** (아직 플랫폼에 없는 작가): 이름 + 이메일로 행 추가 후, "초대 보내기"로 `createExternalArtistAndClaim(projectId=전시)` 호출.  
  - "다음: 작품 선택" 버튼으로 스텝 2로 이동.

- **스텝 2: 작품 선택 (2. exhibition.stepWorks)**  
  - **작품 풀**  
    - **참여 작가를 한 명이라도 선택했을 때**: 선택한 작가들의 **퍼블릭 작품만** `listPublicArtworksByArtistId`로 가져와서 표시 (최대 100개씩).  
    - **참여 작가를 한 명도 선택하지 않았을 때**: **내 작품** + **내가 리스팅한 작품** (`listMyArtworks` + `listPublicArtworksListedByProfileId`, 각 100개).  
  - **필터**  
    - 스텝 2 상단에서 "전체" / 특정 참여 작가 칩으로 **작가 필터** (이미 풀 자체가 위 규칙으로 줄어든 상태).  
    - **텍스트 검색** (제목/매체/스토리/키워드)으로 추가 필터.  
  - **추가 액션**  
    - 작품 카드 "추가" 클릭 → `addWorkToExhibition(id, workId)` 후, 해당 작품에 대해 CURATED 클레임 생성 (전시–큐레이터 프로비넌스).  
  - **새 작품 업로드로 넣기**  
    - "새 작품 업로드" 안내 + **참여 작가별/외부 작가별** `/upload?addToExhibition=전시ID&artistId=...` 등 링크.  
    - 업로드 완료 시 해당 작품은 전시에 자동 추가되고, 유저는 **전시 상세**(`/my/exhibitions/[id]`)로 이동.

- **같은 페이지 상단**  
  - **관리자 위임 초대** (이메일 입력 → delegation invite) 블록이 항상 노출됨.

### 2.2 백 버튼

- 문구: "← Back to My Exhibitions" (`common.backTo` + `exhibition.myExhibitions`)  
- 실제 링크: **`/my/exhibitions/[id]`** (해당 전시 상세)  
→ 라벨은 “내 전시 목록”인데, 동작은 “이 전시로 돌아가기”라서 **일관되지 않음**.

---

## 3. 유저 입장에서의 로직 요약

1. **전시 만들기** → 저장하면 곧바로 **작품 추가** 페이지로 이동 (의도적으로 “먼저 작품 넣기” 유도).  
2. **작품 추가** 페이지는 **2단계**로 보인다.  
   - 1단계: “참여 작가” 고르기 (또는 스킵 가능).  
   - 2단계: “그 작가들의 작품” 또는 “내/내가 리스팅한 작품” 풀에서 고르고 “추가” 클릭.  
3. **참여 작가를 안 고르면**  
   - 풀 = 내 작품 + 내가 리스팅한 작품.  
   - “내 전시에 내 작품만 넣을 때”는 1단계를 건너뛰고 2단계만 쓰면 됨 (탭으로 2 클릭).  
4. **참여 작가를 고르면**  
   - 풀 = 그 작가들의 **퍼블릭** 작품만.  
   - “내가 큐레이터고, 특정 작가들 작품만 전시에 넣을 때” 쓰는 플로우.  
5. **새로 올려서 넣기**  
   - 2단계 안내에서 “업로드” 링크 타고 가서, 업로드 완료 시 자동으로 그 전시에 추가되고, 전시 **상세**로 돌아감.  
   - “한 장 더 넣고 싶다”면 상세에서 다시 “작품 추가” 눌러서 `/add`로 가야 함.

---

## 4. 사용상 불편함·혼동 가능 지점 (유추)

- **1단계가 필수처럼 느껴짐**  
  - 첫 화면이 “1. 참여 작가”라서, “작가를 꼭 골라야 하나?”라고 느낄 수 있음.  
  - “그냥 내 작품만 넣고 싶다”는 유저는 **2. 작품 선택** 탭을 눌러야 한다는 걸 바로 못 찾을 수 있음.

- **참여 작가 = “작품 풀 필터”인데, “참여 작가”라는 말만 보면 “전시에 참여하는 사람 목록”으로 오해 가능**  
  - 실제로는 “지금 이 페이지에서 **어떤 작가의 작품 목록을 볼지**”를 정하는 단계.  
  - “참여 작가에 나를 넣어야 내 작품이 보이나?” 같은 질문이 생길 수 있음.  
  - 참여 작가를 **한 명이라도** 선택하면, **선택한 작가들의 퍼블릭 작품만** 보이므로, “나”를 넣지 않으면 **내 작품이 리스트에서 사라짐**.

- **백 버튼 라벨과 실제 이동**  
  - “Back to My Exhibitions”인데 `/my/exhibitions/[id]`로 가므로, “목록으로 돌아간다”고 생각하고 눌렀다가 전시 상세만 보이면 어색할 수 있음.

- **한 페이지에 역할이 많음**  
  - 같은 화면에 **참여 작가 선택** + **외부 작가 초대** + **관리자 위임 초대** + **작품 선택/추가** + **새 작품 업로드 링크**가 다 있음.  
  - “작품만 넣으려고 왔는데, 초대/위임이 눈에 많이 들어온다”는 반응이 나올 수 있음.

- **새 작품 업로드 후 복귀**  
  - 업로드 완료 시 전시 **상세**로만 가므로, “바로 한 편 더 추가하고 싶다”면 상세에서 다시 “작품 추가”를 눌러야 함.  
  - “작품 추가 페이지로 돌아가기” 옵션이 없어서, 연속으로 여러 개 새로 올리기는 다소 번거로울 수 있음.

- **참여 작가 0명 vs 1명 이상**  
  - 0명: “내 + 내가 리스팅한” 작품.  
  - 1명 이상: “그 작가들 퍼블릭”만.  
  - 이 전환 때문에 “아까 보이던 내 작품이 왜 안 보이지?”라고 느낄 수 있음 (참여 작가를 추가했을 때).

---

## 5. 로직이 “너무 복잡한가?”

- **의도 자체는 읽을 수 있음**  
  - “참여 작가로 풀을 좁히고, 그 풀에서만 골라 넣기” vs “참여 작가 없이 내/리스팅 풀에서 넣기”라는 두 모드가 분리되어 있음.  
- **복잡하게 느껴지는 부분**  
  - 1단계(참여 작가)의 **의미**가 “전시 참여자”가 아니라 “작품 풀 필터”라는 점이 문구만으로는 잘 안 드러남.  
  - “참여 작가 없음 = 내/리스팅”, “참여 작가 있음 = 그 작가들만”이라는 **스위치**가 숨겨져 있어서, 실수로 참여 작가를 하나 넣었다가 “내 작품이 사라졌다”고 느낄 수 있음.  
  - 위임/외부 작가 초대가 같은 페이지에 있어서, “작품만 넣기” 목적이 흐려질 수 있음.

---

## 6. 나중에 개선 시 참고할 수 있는 방향

- **1단계 라벨/설명**  
  - “참여 작가” 대신 “이 전시에 넣을 작품을 **어떤 작가 기준으로 볼지** 선택 (비워두면 내 작품 + 리스팅)” 같은 문구로, “풀 필터”임을 분명히.  
- **백 버튼**  
  - “전시 상세로” / “이 전시로 돌아가기” 등으로 라벨과 링크를 맞추기.  
- **진입 시 기본 스텝**  
  - “참여 작가를 한 명도 선택하지 않은 상태”라면 처음부터 **2. 작품 선택**을 기본 탭으로 두어, “내 작품만 넣는” 케이스를 더 짧게 만들기.  
- **업로드 후 복귀**  
  - “한 편 더 추가” 시 전시 상세가 아니라 `/add`로 돌아오는 옵션을 주면, 연속 추가가 덜 번거로울 수 있음.

이 문서는 “지금 전시 게시물 업로드 후 작품 추가”가 **어떤 경로·로직으로 되어 있는지**와, 그로 인해 **생길 수 있는 사용상 불편함**을 정리한 점검 결과다.
