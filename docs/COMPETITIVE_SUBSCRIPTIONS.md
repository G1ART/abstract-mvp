## 2) Initial benchmarks (seed notes)

### 2.1 LinkedIn Premium (career network)
- Category: Professional social network
- Primary buyer: Individuals (job seekers / sales / recruiters)
- Core “why pay”: 인사이트(누가 봤는지), 더 강한 탐색/리치(필터/메시징)
- Free value: 네트워크/피드/기본 검색
- Paid value: “Who viewed your profile”, advanced filters, InMail 등
- Paywall placement: 프로필 방문자 리스트/고급 검색/메시지 영역
- Upsell surfaces: 인사이트 카드, 검색 필터 사용 시
- Abstract mapping: Artist Pro = viewers/engagers + network composition, Discovery Pro = advanced people/artwork filters

### 2.2 SoundCloud (creator audio)
- Category: Creator platform + distribution tools
- Primary buyer: Creators
- Core “why pay”: 업로드/배급/분석/프로모션(커리어 레벨업 툴)
- Free value: 기본 업로드/재생
- Paid value: 고급 분석, 업로드 확장, 배급/수익화, promotion
- Upsell surfaces: 업로드 제한, 분석 페이지, 성장 기능
- Abstract mapping: Artist Pro = 운영툴(배치/컬렉션/대표작) + 인사이트 + 노출부스트

### 2.3 Patreon (creator memberships; take rate)
- Category: Creator monetization membership
- Primary buyer: Fans/Patrons; platform takes %
- Core “why pay”: 팬 구독으로 창작자 지원, 독점 콘텐츠
- Free value: 공개 콘텐츠
- Paid value: 구독자 전용 콘텐츠/혜택
- Upsell surfaces: 콘텐츠 중간/끝, creator page
- Abstract mapping: (Future) 작품/프라이빗 룸/collector circle 같은 “멤버십” 가능성

### 2.4 Substack (publishing paywall)
- Category: Paid newsletters
- Primary buyer: Readers
- Core “why pay”: 유료 콘텐츠 접근(프리뷰→락)
- Upsell surfaces: 프리뷰 끝/중간 “unlock”
- Abstract mapping: Discovery Pro = price/extra fields/private works unlock (artist-controlled)

### 2.5 Behance / ArtStation (portfolio + marketplace-ish)
- Category: Portfolio discovery
- Primary buyer: Creators (promotion), recruiters/clients
- Core “why pay”: 노출/프로필 강화/마켓 기능(플랫폼마다 상이)
- Abstract mapping: People/추천/탐색이 곧 가치. “전체 공개”만으론 수익화가 약하므로 인사이트/락/툴이 중요.

---

## 3) Abstract synthesis (current hypothesis)
- Artist Pro: **Curiosity + leverage**
  - viewers/engagers list, persona composition, exposure boost, pro management tools
- Discovery Pro: **Access + efficiency**
  - price/extra fields unlock, private access requests, advanced filters/saved searches
